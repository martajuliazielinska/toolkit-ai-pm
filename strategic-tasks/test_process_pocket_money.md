# Pocket Money Flow

## Steps

1. **Parent Setup** — Parent creates pocket money account for child, sets allowance amount and frequency (daily/weekly/monthly)
2. **Allowance Trigger** — System automatically distributes allowance on scheduled day at fixed time
3. **Task Assignment** — Parent creates optional tasks (chores) that child can complete for bonus rewards
4. **Child Completes Task** — Child marks task as done, uploads photo proof (optional)
5. **Parent Approval** — Parent reviews task completion, approves or rejects, logs decision
6. **Reward Distribution** — If approved, reward amount transferred to child account
7. **Child Spending** — Child views balance, can spend from virtual wallet (no external payments yet)
8. **Savings Goal** — Child sets personal savings goal, system tracks progress toward target
9. **Goal Achievement** — Child reaches goal amount, parent can unlock access to goal funds or extend deadline
10. **Transaction History** — Child views all transactions (allowance, rewards, spending) with parental visibility
